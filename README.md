# Smart Pet Feeder (YOLO + Raspberry Pi)

## Description
AI-powered pet feeder using facial recognition to identify pets and dispense food.

## Features
- Pet detection using YOLO
- Automatic feeding system
- Web dashboard (Flask)
- Feeding history tracking

## Technologies
- Python
- OpenCV
- YOLO (NCNN)
- Raspberry Pi 5

## How to Run
1. Install dependencies:
   flask
   opencv-python
   numpy

2. Run:
   python3 dashboard.py
